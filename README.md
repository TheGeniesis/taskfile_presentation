# Taskfile presentation

This repo contains taskfile examples created for the presentation

## Links

- [Taskfile](https://taskfile.dev/)
- [Taskfile vs GNU make article](https://tsh.io/blog/taskfile-or-gnu-make-for-automation/)

## Structure

```shell
task_presentation
├── Scripts # Additional scripts
├── .env # File with environments variables
├── README.md # this readme file
└── Taskfile.yml # the main file with tasks to execute
└── Taskfile-install.yml # additional scripts to download binaries
```
